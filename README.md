#  Installation
-1. Clone the repository
git clone `https://github.com/Kirogii/MangaAMTL.git`
cd MangaAMTL
-2. Install dependencies
`pip install`
-3. Run the development server
`python app.py"
-4
The application will start on:
http://localhost:7860
