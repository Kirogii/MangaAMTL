#  Installation
- 1: Clone the repository
`git clone https://github.com/Kirogii/MangaAMTL.git`
- 2: Cd inside
`cd MangaAMTL`
- 3: Install dependencies
`pip install`
- 4: Run the development server
`python app.py`

` The application will start on:
 http://localhost:7860`
